# Curves Generator

## Core level methods

### Promoted

```python
RevertToDefault()
```
Set default curves.

```python
SelectCurve(curveType)
```
Set curve generator to given type.

```python
WriteValues(param, value)
```
Set given values to param.

### Private

```python
getCurrent()
```
Return current settings for curve.

```python
modifyName(par, name)
```
Change param name.

```python
updateUI(a, b, c)
```
Update UI.
